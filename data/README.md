# Data

## `raw/`
Hand-entered measurements and constants **exactly as recorded** in the lab notebook.

Files:

- `screen_angle.csv`  
  Screen distance **L** and order separation **y** for the screen-angle method (Experiment 1(i)).

- `camera_calibration.csv`  
  Camera calibration spacing and pixel counts (Experiment 1(ii)).

- `slit_cutoff.csv`  
  Variable slit cutoff width and instrument constants (Experiment 1(iii)).

- `abbe_params.csv`  
  Parameters used to estimate NA and Abbe/Rayleigh resolution limit (Experiment 2).

## `processed/`
Generated by `analysis/analyze.py` (do not edit by hand).

- `results.json` – full structured output
- `grating_results.csv` – summary table for Experiment 1
